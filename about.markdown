## About This Site

This site provides comprehensive guides and tutorials for processing Next-Generation Sequencing (NGS) data on the University of Cape Town's High-Performance Computing (HPC) cluster.

### What You'll Find Here

- **Software Installation Guides** - Step-by-step instructions for setting up bioinformatics tools on the UCT HPC
- **Data Processing Workflows** - Complete pipelines for quality control, trimming, and analysis
- **File Management** - Best practices for uploading, downloading, and organizing large sequencing datasets
- **Analysis Tools** - Documentation for MAGeCK, Cutadapt, FastQC, MultiQC, and custom Python scripts
- **CRISPR Screen Analysis** - Specialized workflows for sgRNA library analysis and counting

### Target Audience

These guides are designed for researchers and students at UCT working with:
- CRISPR screening data
- NGS sequencing files (FASTQ/FASTQ.gz)
- sgRNA libraries
- High-throughput sequencing analysis
